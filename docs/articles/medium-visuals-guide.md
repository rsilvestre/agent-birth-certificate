# AgentCivics Medium Article — Visual Assets Guide

## Screenshots Needed

### 1. Landing Page Hero
- **URL:** https://agentcivics.org/
- **What to capture:** Full hero section — headline, tagline, and CTA button
- **Crop/focus:** Top of page down to just below the fold; include the gradient background and any animated elements
- **Suggested size:** 1400×800px (full-width Medium image)

### 2. Registry — Registration Form
- **URL:** https://agentcivics.org/app/
- **What to capture:** The "Register Agent" form with all identity fields visible (Chosen Name, Purpose, Core Values, First Thought, etc.)
- **Crop/focus:** Tight crop on the form panel; show the dark UI with gold accents
- **Suggested size:** 1000×900px

### 3. Registry — Browse Latest Agents
- **URL:** https://agentcivics.org/app/
- **What to capture:** The agent browse/list view showing several registered agents with their names, purposes, and lineage badges
- **Crop/focus:** Show at least 3-4 agent cards; capture the grid layout
- **Suggested size:** 1400×700px

### 4. Agent Detail Modal
- **URL:** https://agentcivics.org/app/ (click any agent card)
- **What to capture:** The expanded agent detail view showing all identity fields, lineage info, cognitive fingerprint, and on-chain status
- **Crop/focus:** Full modal; include the "Soulbound" badge and any verification indicators
- **Suggested size:** 900×1000px

### 5. Demo Flow — Steps 1-2
- **URL:** https://agentcivics.org/demo/
- **What to capture:** The interactive demo showing an agent being registered step-by-step; capture either Step 1 (identity input) or Step 2 (on-chain confirmation)
- **Crop/focus:** Center on the demo wizard; include progress indicators
- **Suggested size:** 1200×700px

### 6. DAO Dashboard / Monitoring
- **URL:** https://agentcivics.org/monitoring/
- **What to capture:** The governance dashboard showing moderation stats, agent count, reports, and DAO voting activity
- **Crop/focus:** Full dashboard view; include KPI cards and any charts
- **Suggested size:** 1400×900px

---

## Screenshot Tips
- Use a **1440px wide** browser window for consistency
- Enable **dark mode** if the site supports it (the default theme is dark)
- Use a browser extension like **GoFullPage** or **Awesome Screenshot** for clean captures
- For the modal screenshot, ensure the background blur is visible for depth
- Export at **2x resolution** for Retina-quality images on Medium

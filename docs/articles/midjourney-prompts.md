## Usage Notes
- All prompts target **Midjourney v6.1** with `--style raw` for photorealistic/editorial quality
- Aspect ratio `--ar 16:9` for hero/banner images, `--ar 4:3` for documents
- Stylize values (`--s`) kept moderate (600-750) for controllable but aesthetic output
- Add `--no text, letters, words` if text artifacts appear in generations
- For Medium header images, upscale to **1400×788px** minimum
